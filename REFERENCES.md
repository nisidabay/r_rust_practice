# References

Books and resources for learning Rust. Listed for discoverability — pick the ones that match your learning style.

## Core Rust

| Book | Author | Why |
|------|--------|-----|
| **The Rust Programming Language** | Klabnik & Nichols | The official book. Free online, covers everything from hello to async. |
| **Rust by Example** | Rust Community | https://doc.rust-lang.org/stable/rust-by-example/ — code-first, minimal prose. |
| **Programming Rust (2nd Ed.)** | Blandy, Orendorff, Tindall | Deep systems-level explanation. Good after you've done the basics. |
| **Rust for Rustaceans** | Gjengset | Intermediate/advanced: async, unsafe, design patterns. Read after you've shipped something. |

## Online

| Resource | URL | Why |
|----------|-----|-----|
| Rust Playground | https://play.rust-lang.org/ | Run snippets without installing |
| Rustlings | https://github.com/rust-lang/rustlings | Interactive exercises (great companion to this repo) |
| Rustonomicon | https://doc.rust-lang.org/nomicon/ | Unsafe Rust — the dark arts |
| Rust Edition Guide | https://doc.rust-lang.org/edition-guide/ | Edition migration, edition 2021 features |
| std docs | https://doc.rust-lang.org/std/ | The standard library reference (bookmark this) |
| Cargo docs | https://doc.rust-lang.org/cargo/ | Package manager, workspaces, profiles |
